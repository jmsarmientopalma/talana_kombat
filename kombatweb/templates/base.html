Instrucciones de uso:

Para cargar las instrucciones de pelea en el programa, debe dejar el archivo JSON dentro del directorio kombatweb/static/json/ antes de continuar.


Formato JSON:

El JSON a cargar debe estar en el formato adecuado. De lo contrario, el programa no procesara la pelea. El formato requerido es el siguiente:

{'player1': {'movimientos':[lista-de-movimientos], 'golpes':[lista-de-golpes]},'player2': {misma-estructura-player1}}'
